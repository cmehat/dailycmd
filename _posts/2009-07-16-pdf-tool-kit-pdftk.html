--- layout: post title: "PDF tool kit : pdftk" date: 2009-07-16 11:56:00 +0000 permalink: /2009/07/pdf-tool-kit-pdftk.html tags: ["pdf"] author: "cm" --- {% raw %} pdftk is a toolkit available for example on ubuntu repositories. It seems to be from the "PDF Hacks" book.


Main Webpage :
http://www.accesspdf.com/pdftk/


List of the features found on the website (Latest Version: 1.41, Released: November 28, 2006) :

  • Merge PDF Documents
  • Split PDF Pages into a New Document
  • Rotate PDF Pages or Documents
  • Decrypt Input as Necessary (Password Required)
  • Encrypt Output as Desired
  • Fill PDF Forms with FDF Data or XFDF Data and/or Flatten Forms
  • Apply a Background Watermark or a Foreground Stamp
  • Report on PDF Metrics such as Metadata, Bookmarks, and Page Labels
  • Update PDF Metadata
  • Attach Files to PDF Pages or the PDF Document
  • Unpack PDF Attachments
  • Burst a PDF Document into Single Pages
  • Uncompress and Re-Compress Page Streams
  • Repair Corrupted PDF (Where Possible)

--- layout: post title: "Find file encoding" date: 2007-05-18 11:49:00 +0000 permalink: /2007/05/find-file-encoding.html author: "cm" --- {% raw %} Trouver l'encodage d'un fichier :

file --mime filename
Par exemple :
$ file filename.tex
filename.tex: LaTeX 2e document text
cm$ file --mime filename.tex
filename.tex: text/plain; charset=iso-8859-1
{% endraw %}
